# Football Score Predictor (frontend)

### Prerequisites

To run the api, please install:
* falcon
```
pip install falcon
```

* Gunicorn
```
pip install gunicorn
```

To run the api:
* Go into api folder
* run following:
```
gunicorn app
```

Make sure you are running by going to http://127.0.0.1:8000/get_score in browser
